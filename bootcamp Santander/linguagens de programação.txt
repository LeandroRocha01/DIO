	linguagem de programação
método padronizado de implementação de um código fonte
conjunto de regras e palavras reservadas para cada linguagem de modo que o computador e o programador entendam

	código fonte
código gerado através do uso da linguagem de programação(de alto nível)

Pode ser: 
- Traduzido
o programa compilador converte o código para uma linguagem de baixo nível(linguagem de máquina)
   para que o computador interprete corretamente
a versão código convertido é chamada de programa objeto

- Interpretado 
não precisa ser traduzido, a máquina entende o código de baixo nível
tende a ser um processo mais demorado


	Caracteristicas de um programa

diretrizes que devem ser consideradas antes de desenvolver um sistema:
-legibilidade
	deve ser legível, interpretável pela máquina e por outros porgramadores
	codigo seja coerente
-redigibilidade
	bem desenvolvido, no sentido de boas práticas e sintaxe
	fácil escrita, sem complexidade desnecessaria
	reutilizar tudo q for possível
-confiabilidade
	executar oque se propos a fazer
	identificar e demonstrar os erros
	compatibiidade entre máquinas
-custo
	usar os recursos concientemente
	sem desperdício
	consumir o minimo de recurso computacional possivel
